# Portfolio Site (Retro Gadget Style)

かわいげのあるレトロガジェット（ファミコンや初期Mac）風のポップなテックデザインを採用した、個人ポートフォリオサイトです。

##  特徴
- **デザイン**: アイボリーをベースカラーとした温かみのあるレトロポップな世界観
- **エフェクト**: CSSフィルターを用いた、画像へのドット絵（ピクセルアート）風エフェクトの実装
- **URL構造**: ユーザーフレンドリーな Pretty URLs (`/about/`, `/works/`) の採用

##  ディレクトリ構造
```text
/
├── index.html             # トップページ
├── about/
│   └── index.html         # プロフィール詳細ページ
├── works/
│   └── index.html         # 制作実績の詳細解説ページ
├── components/            # 共通パーツ（header, footer, contactなど）
├── css/
│   └── style.css          # スタイルシート（ドット絵風エフェクト含む）
├── js/
│   └── main.js            # メインスクリプト
└── images/                # 最適化済み画像フォルダ
```
